# Contributing to otis-core

We welcome contributions from the defense and robotics communities. To maintain our dual-licensing model (GPLv3 / Commercial), all contributors must agree to the SkyGuard Contributor License Agreement (CLA).

By submitting a Pull Request, you grant SkyGuard Defense Systems a perpetual, worldwide, non-exclusive, no-charge, royalty-free, irrevocable copyright license to reproduce, prepare derivative works of, publicly display, publicly perform, sublicense, and distribute your contributions.

**To Contribute:**
1. Fork the repository.
2. Create a feature branch (`git checkout -b feature/advanced-kinematics`).
3. Commit your changes.
4. Open a Pull Request and sign the CLA prompt in the PR comments.
